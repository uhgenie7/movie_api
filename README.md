## 사이트

- [메인페이지](http://uhj1993.dothome.co.kr/movie_api)


## 주요 특징
- 네이버 오픈 API로 영화 검색 구현
- yts가 제공해주는 API를 받아 영화 리스트 구현 및 상세 정보 페이지

- [CSS3]
